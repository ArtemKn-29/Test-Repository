У даному репозиторії зібрані всі виконані завдання з квест-практики.
Матеріали структуровані у відповідних папках для зручної навігації та перевірки.

Кожна папка містить файли, що відносяться до конкретного завдання, а також додаткові матеріали, такі як:
вихідні файли;
посилання на зовнішні ресурси (за потреби);
скріншоти, які підтверджують виконання або демонструють результат роботи.

Репозиторій створений з метою зберігання, систематизації та наочного представлення результатів проходження квест-практики
